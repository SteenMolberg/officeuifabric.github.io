Features

View all the features available in the Office UI Fabric. While the site is in progress the icons in the navigation to indicate if the work is pending or completed. Items not yet complete will point to the corresponding issue in the GitHub list.

Animations

Use CSS classes to use the CSS animations provided in the Office UI Fabric that you see in familiar Office products.

Color

Use CSS classes to use 9 theme colors & 11 neutral colors. Each one has helper classes for text, border, background and hover states.

Icons

Use CSS classes to use some of the nearly 400 icons provided in the Office UI Fabric that you see in familiar Office products.

Localization Support

The Office UI Fabric includes lots of support for localization such as right-to-left support and language-optmized fonts among others.

Responsive Grid

Use CSS to easily create flexible and responsive layouts using the popular grid-12 system.

Typography

Use CSS classes to apply different font styles including extra font decorations like light to bold.
